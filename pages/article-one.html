Networking 101: From Connections to Security

Computer Networking

The process of bridging two or more computing devices to facilitate the communication of information and resources though physical or wireless mediums (“What is computer networking?”, 2024) (“SMB University: Selling Cisco,” n.d.). When these devices have indeed formed connections between each other, we call this a network.

While there are many different types of networks, two basic setups will be discussed:

Local Area Network (LAN) – a collection of connected devices in one physical location (“What is computer networking?”, 2024) (“What is Computer Networking?”, n.d.) (“SMB University: Selling Cisco,” n.d.). A LAN can vary in size and include both wired and wireless devices (“What is computer networking?”, 2024) (“What is Computer Networking?”, n.d.) (“SMB University: Selling Cisco,” n.d.).

Wide Area Network (WAN) – a collection of LANs and/or individual devices that stretches over a large geographical area (“What is computer networking?”, 2024) (“What is Computer Networking?”, n.d.) (“SMB University: Selling Cisco,” n.d.). WANs are typically used by large organizations and government bodies (“What is computer networking?”, 2024) (“What is Computer Networking?”, n.d.) (“SMB University: Selling Cisco,” n.d.).

For any network, there are three basic network devices that must be included to provide proper connection between devices:

Switch – this device establishes the connection between devices within a network and act as a controller in a LAN (“What is Computer Networking?”, n.d.).

Router – this device connects multiple networks together and to the Internet. It acts as a forwarder that chooses the best path to take to send the information it receives (“What is Computer Networking?”, n.d.).

Access Point (AP) – this device allows wireless to create a connection with a network without the use of cables. It acts as an amplifier where a network’s wireless signal is transmitted across an area to wireless clients (“What is Computer Networking?”, n.d.).

OSI 7-Layer Model

The Open Systems Interconnection (OSI) 7-Layer Model is a theoretical structure that separates network communication into seven layers (“What is OSI Model?”, n.d.). Since sending data over a network is quite complex, this framework makes clear divides in the networking process and provides a universal language for network administrators, designers, IT professionals, and various technologies (“SMB University: Selling Cisco,” n.d.) (“What is OSI Model?”, n.d.). Every layer depends on technologies, protocols, and methods used in the previous layer, with Layer 1 (Physical) being the foundation. Not all systems that utilize the OSI model apply every layer (“What is OSI Model?”, n.d.).

OSI 7-Layer Model (“SMB University: Selling Cisco,” n.d.) (“What is OSI Model?”, n.d.)

Layer Explanation
7. Application Layer The applications themselves and their communication methods (e.g., web browsers using HTTP/HTTPS).
6. Presentation Layer The formation of data used in the Application layer (e.g., HTML or CSS code, which websites use to display information over the Internet).
5. Session Layer The network coordination between applications (e.g., NFS or SMB protocols, which establish, synchronize, and end communication between application hosts).
4. Transport Layer The transmission of data packets in the right order, at the proper rate, and with no errors (e.g., TCP and UDP protocols to send packets from one host to another).
3. Network Layer The connection, routing, forwarding, and addressing of data packets across a network (e.g., IPv4 and IPv6 protocols to send data frames between two IP addresses).
2. Data Link Layer The formatting of data into frames and transmission of those frames across physical mediums (e.g., sending data between hosts’ MAC addresses using Ethernet).
1. Physical Layer The physical mediums and technologies used to send raw data in bits (1’s and 0’s) (e.g., fibre-optic cables, copper cabling, Wi-Fi, Bluetooth, etc.).
