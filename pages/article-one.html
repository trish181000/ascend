Networking 101:

From Connections to Security

Computer Networking

The process of bridging two or more computing devices to facilitate the communication of information and resources though physical or wireless mediums. When these devices have indeed formed connections between each other, we call this a network.

While there are many different types of networks, two basic setups will be discussed:

Local Area Network (LAN) – a collection of connected devices in one physical location.A LAN can vary in size and include both wired and wireless devices.

Wide Area Network (WAN) – a collection of LANs and/or individual devices that stretches over a large geographical area.

For any network, there are three basic network devices that must be included to provide proper connection between devices:

Switch – this device establishes the connection between devices within a network and act as a controller in a LAN.

Router – this device connects multiple networks together and to the Internet. It acts as a forwarder that chooses the best path to take to send the information it receives.

Access Point (AP) – this device allows wireless to create a connection with a network without the use of cables. It acts as an amplifier where a network’s wireless signal is transmitted across an area to wireless clients.

OSI 7-Layer Model

The Open Systems Interconnection (OSI) 7-Layer Model is a theoretical structure that separates network communication into seven layers. Since sending data over a network is quite complex,this framework makes clear divides in the networking process and provides a universal language for network administrators, designers, IT professionals, and various technologies. Every layer depends on technologies, protocols, and methods used in the previous layer, with Layer 1 (Physical) being the foundation. Not all systems that utilize the OSI model apply every layer.

OSI 7-Layer Model

Layer Explanation
7. Application Layer The applications themselves and their communication methods (e.g., web browsers using HTTP/HTTPS).
6. Presentation Layer The formation of data used in the Application layer (e.g., HTML or CSS code, which websites use to display information over the Internet).
5. Session Layer The network coordination between applications (e.g., NFS or SMB protocols, which establish, synchronize, and end communication between application hosts).
4. Transport Layer The transmission of data packets in the right order, at the proper rate, and with no errors (e.g., TCP and UDP protocols to send packets from one host to another).
3. Network Layer The connection, routing, forwarding, and addressing of data packets across a network (e.g., IPv4 and IPv6 protocols to send data frames between two IP addresses).
2. Data Link Layer The formatting of data into frames and transmission of those frames across physical mediums (e.g., sending data between hosts’ MAC addresses using Ethernet).
1. Physical Layer The physical mediums and technologies used to send raw data in bits (1’s and 0’s) (e.g., fibre-optic cables, copper cabling, Wi-Fi, Bluetooth, etc.).

Definitions

HTTP/HTTPS – Hypertext Transfer Protocol / Hypertext Transfer Protocol Secure are layer 7 protocols that facilitate communication between two or more web clients.

NFS – Network File System is a file sharing solution that allows two or more computers to access and transfer files.

SMB – Server Message Block is a file sharing solution that allows computer applications to read and write to files.

TCP – Transmission Control Protocol is a connection protocol that establishes connection between applications before sending data.

UDP – User Datagram Protocol is a connection protocol that establishes a fast but unreliable connection between applications and is typically used when reliability is not required.

IPv4/IPv6 – A standard which defines in which the network layers of two devices communicate.

IP Address – A 32-bit logical addressing scheme.

MAC Address – Media Access Control addresses are unique hardware identifiers that are protocol-independent.

Ethernet – A connection technology that operates at layers 2 and 3 of the OSI model. It allows two or more devices to communicate with each other via physical cabling.

Fibre-optic – A connection technology that sends data in the form of pulses of light through threads of glass.

Wi-Fi – A wireless connection technology that uses radio waves to transmit data between two or more devices through frequencies.

Bluetooth – A short-range wireless connection technology that sends data between two or more devices in a peer-to-peer connection and without the need for wireless routers or access points.

Network Security

Network Security is the design and practice of protecting a network; this includes its data, resources, clients, connection methods, usability, and accessibility. It achieves this using hardware and software technologies, methods, protocols, policies, and controls. By combining multiple layers of this security, a network is provided with comprehensive protection from unauthorized access, hackers, and vulnerabilities; this is also known as Defense in Depth. Examples of network security include firewalls, email security, anti-virus and anti-malware software, network segmentation, access control, application security, and data loss prevention.

Defense in Depth – An information security strategy that utilizes people, technology, and operations to create protective barriers across multiple layers of a network.
